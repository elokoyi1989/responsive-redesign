Project Overview

This portfolio documents the responsive redesign of the Berkshire Hathaway homepage. The goal was to retain the brand’s no-frills aesthetic while adding bold, engaging visuals, improved usability, and accessibility across all devices.

Design Process

Key stages:

  • Usability & accessibility audit of the original site
  • Wireframing responsive layouts
  • Visual style guide (vibrant accents, dynamic type, component states)
  • High-fidelity mockups for mobile, tablet, desktop
  • HTML/CSS implementation & responsive testing
